idea vim 配置

Published: Mon 14 August 2023
By junteng

In tools.

tags: idea vim

idea的vim配置文件是.ideavimrc，和vim的.vimrc大同小异 大致可以分成下面几个部分

set

这个和vim的配置文件一样:

set nu
set tabstop
set expandtab
set smartcase
set ignorecase
set incsearch
set hlsearch
但是有几个是idea特有的:
set clipboard=ideaput
set ideamarks
set ideajoin

启用插件

这个需要在idea的plugins里面安装，一般搜索vim就可以了

set NERDTree 
set surround
set easymotion
set highlightedyank
set sneak
set which-key "自动切换输入法，windows需要安装英语

键位映射

map只能配合<action>使用，括号里面是idea的各种命令，可以使用:actionlist "你搜索的命令"查找,也可以打开ideavim的action track，来查找idea到功能
map <leader>i <action>(GotoImplementation)
map <leader>b <action>(GotoDeclaration)
map <leader>q <action>(CloseContent)
map <leader>w <action>(SaveAll)
map <leader>c <action>(GotoClass)
map <leader>f <action>(GotoFile)
map <leader>a <action>(GotoAction)
map <leader>s <action>(GotoSymbol)
map <leader>h <action>(TypeHierarchy)

nmap,nnoremap,imap,inoremap等，可以映射到:命令<CR>来使用
nmap gh :action ShowErrorDescription<cr>
nmap gq :action ReformatCode<CR>
nmap gQ :action ShowReformatFileDialog<cr>
nmap <leader>p :action ParameterInfo<cr>

noremap H ^
noremap L $
nnoremap Y y$
nnoremap <leader>n :NERDTreeFocus<cr>
nnoremap <leader>t :NERDTree<cr>
nnoremap <leader>r :RenameElement<cr>

处理idea和vim到冲突

v代表可视模式，i代表插入模式，n代表normal模式，a代表所有模式
sethandler <c-c> v:ide
sethandler <c-v> i:ide
sethandler <c-x> v:ide
sethandler <c-j> i:ide
"sethandler <c-p> i:ide
sethandler <c-e> n:ide
sethandler <c-a> n:ide
